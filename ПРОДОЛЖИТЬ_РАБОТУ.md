## Как работать (инструкция для ИИ)

1. **Перед кодом** — прочитать `СЛЕДУЮЩИЕ_ЗАДАЧИ.md` и **grep/read**, что задача ещё не сделана.
2. **За один срез** — 5 prod-задач (P1 код), без GPU/E2E если не просят явно.
3. **После среза:**
   - дописать блок в конец `чтоготовосегодня.md`;
   - обновить `СЛЕДУЮЩИЕ_ЗАДАЧИ.md` (5 новых, без дубликатов);
   - pytest slice + `flutter gen-l10n` где нужно;
   - **commit + push на `main`** (пользователь просит пушить).
4. **Стиль:** минимальный diff, существующие паттерны, slice-тесты `test_*_slice.py`.
5. **PowerShell:** `;` вместо `&&`; commit message простой `-m "..."`.
