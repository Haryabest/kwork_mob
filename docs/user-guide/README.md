# Руководство пользователя

## Быстрый старт

1. Скачайте приложение (iOS / Android)
2. Зарегистрируйтесь по email (ФИО не обязательно)
3. Нажмите «Снять товар» → выберите категорию
4. Снимите 12 ракурсов (ручной спуск, AR-подсказки)
5. Оплатите тариф (2 990 / 5 990 ₽)
6. Скачайте .glb / .usdz и опубликуйте на WB/Ozon

## Тарифы

| Тариф | Цена | Объект |
|-------|------|--------|
| Малый | 2 990 ₽ | до 30 см |
| Крупный | 5 990 ₽ | свыше 30 см |

## Корпоративный режим

Owner может приглашать Photographer, Manager, Viewer.
Съёмка по ссылке — для внештатных фотографов без регистрации.
