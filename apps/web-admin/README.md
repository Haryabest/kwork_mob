# Staff Panel (бывш. web-admin + web-support)

Единая внутренняя панель для **владельца сервиса** и **специалистов поддержки**.

| Роль | JWT role (backend) | Доступ |
|------|-------------------|--------|
| Владелец | `admin` | всё: воркеры, финансы, B2B, модерация, настройки + поддержка |
| Поддержка | `support_agent` | только обращения и FAQ |

Порт dev: **3001**

```bash
cd apps/web-admin && npm run dev
```

> `apps/web-support` удалён — функционал перенесён сюда.
