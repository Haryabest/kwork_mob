# Web Seller — личный кабинет селлера

**Не путать с Staff Panel (`web-admin`)!**

Это сайт для **клиентов сервиса** — продавцов на Wildberries/Ozon.

## Где в ТЗ

- **§2.4** — единый личный кабинет (мобилка + веб)
- **§20** — полная спецификация веб-сайта селлера
- **§8.12** — пополнение баланса через веб

## Для кого

| Пользователь | Что делает на сайте |
|--------------|---------------------|
| Физлицо-селлер | Баланс, история моделей, настройки |
| Owner компании | + управление командой, лимиты, API-ключи |
| Manager / Viewer | Ограниченный доступ по ролям компании |

## Чего здесь НЕТ

- Управления воркерами, промокодами платформы, NSFW-модерации — это **Staff Panel** (`web-admin`)
